
# 🎓 Student Database Management System (C)

A menu-driven Student Database Management System developed in C using a Singly Linked List.

## Features

- Insert Student at Beginning
- Insert Student at End
- Insert Student at Middle
- Display Student Records
- Search Student
- Delete Student
- Sort Students by Roll Number
- Rotate Linked List (Left/Right)
- Display Topper
- Display Failed Students
- Save Records to File
- Read Records from File
- Duplicate Roll Number Validation

## Data Structure Used

- Singly Linked List

## Technologies

- C Programming
- File Handling
- Dynamic Memory Allocation
- Linked List

## Compilation

```bash
gcc student_database.c -o student_database
./student_database
```
