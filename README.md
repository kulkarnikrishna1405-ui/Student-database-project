# 🎓 Student Database Management System

> A **menu-driven Student Database Management System** built in **C** using **Singly Linked Lists** and **File Handling**. This project demonstrates efficient data management through dynamic memory allocation and fundamental data structure operations.

---

## ✨ Features

✅ Add Student Records  
✅ Display Student Records  
✅ Modify Records (Roll No / Name / Marks)  
✅ Delete Records (Roll No / Name)  
✅ Sort by Name & Marks  
✅ Reverse Linked List  
✅ Save & Load Records using Files  
✅ Delete All Records  
✅ Input Validation & Duplicate Name Handling

---

## 🛠️ Tech Stack

- 💻 C Programming
- 🔗 Singly Linked List
- 📂 File Handling
- 🧠 Dynamic Memory Allocation
- 🎯 Structures & Pointers

---

## 🚀 Getting Started

### Compile

```bash
gcc student_database.c -o student_database
```

### Run

```bash
./student_database
```
